# NestJS-Admin-API
> Handler RESTful API for admin-dashboard

* Authorization - casbin
* Authentication - PassportJS
* Environment - dotenv
* Validation - class-validator
* Logging - winston
* Monitor - Prometheus
* APM - ELK
* ORM - TypeORM v3.x

## How To Use
```zsh
$ cp .env.example .env
$ docker compose up -d
$ npm run start:dev
```
